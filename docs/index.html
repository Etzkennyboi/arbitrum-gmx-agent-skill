API Reference

24 endpoints available — both Node.js module and HTTP REST API

📊 GMX Trading (18 endpoints)

GET /prices/:pair ETH, BTC, ARB, SOL, LINK price (Chainlink)
GET /markets All 5 GMX markets + open interest
POST /position/long Open leveraged long position
POST /position/short Open leveraged short position
GET /analyze/:market Strategy recommendation (LEAN_LONG / LEAN_SHORT / NEUTRAL)
GET /funding/:market Funding rate signals
+ 12 more... Pool stats, wallet summary, monitoring, identity, etc.

⚡ Liquidation Hunter (6 endpoints)

GET /liquidation/check/:account Check if account is liquidatable (health factor < 1)
POST /liquidation/calculate Calculate exact profit with gas + slippage
GET /liquidation/opportunities Scan all liquidatable accounts
POST /liquidation/simulate Dry-run liquidation (no gas cost)
POST /liquidation/execute Execute liquidation (requires private key)
GET /health Server health + current block

Example: Execute Liquidation

curl -X POST http://localhost:3000/liquidation/execute \
  -H "Content-Type: application/json" \
  -d '{
    "targetAccount": "0x7a94...Cf8e",
    "debtAsset": "0xaf88d065e77c8cC2239327C5EDb3A5601c834C59",
    "collateralAsset": "0x82aF49447d8a07e3bd95bd0d56f313D432751f2D",
    "debtAmount": "100000000000"
  }'
